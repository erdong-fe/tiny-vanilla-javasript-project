# tiny-vanilla-javasript-project
### 这个项目是什么：
使用原生JavaScript实现的小项目，锻炼前端编程实战能力
不使用任何框架（vue/react），就纯js、css、html，非常适合新手入门，增强编程能力
也适合已经入门但是没有项目练手的前端
